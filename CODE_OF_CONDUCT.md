# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its Code of Conduct.

## Reporting

If you experience or witness behavior that violates this Code, please email:

- **conduct@loomantix.com** (for issues involving Loomantix maintainers or the project as a whole)
- The PR/issue reporter directly (for low-stakes interpersonal misunderstandings that don't rise to a CoC violation)

Reports are reviewed by the project maintainers. Confidentiality is preserved unless disclosure is required to protect the safety of the community.

## Enforcement

Maintainers may take any action they deem appropriate, including:

- Warning the offender privately
- Requiring a public or private apology
- Temporarily or permanently revoking commit access, repository access, or community privileges
- Reporting to the offender's employer or other authorities for illegal behavior

Decisions made by the maintainers are final.
